#RdioCommander 0.1
A tool for monitoring the Rdio desktop application. Built in Python, and using OSASCRIPT to handle interaction with Rdio.

##Setup
> $ python setup.py install

##Usage
    $ rdio -h
    RDIO COMMANDER 0.1
    Rdio.py -d <delay>
    Rdio.py -c <light | dark>
