RdioCommander
=============

A Python and AppleScript tool for monitoring the OSX Rdio application.


                                                 NOW PLAYING ON RDIO
                                                 --------------------
                                                   Authority Zero
                                                   One More Minute
                                                   
                                              
##Instructions
> python bin/rdio

By default this refreshes every 10 seconds, to customize that try the following


> python bin/rdio -d 5
